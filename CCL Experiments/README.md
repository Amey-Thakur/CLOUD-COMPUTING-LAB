> [!TIP]
> **Cloud Services & AWS**: When working with AWS services, always follow the principle of least privilege for IAM permissions. Ensure you clean up resources after experiments to avoid unexpected charges. Use the AWS Free Tier effectively for learning purposes.

> [!WARNING]
> **Cloud Deployment**: Ensure your static websites are correctly configured for public access in S3 buckets. Always verify your bucket policies and CORS settings to prevent access issues. Since these experiments involve live cloud infrastructure, monitor your AWS usage regularly.

## How to Use

### Viewing Reports
1. **Navigate** to the specific experiment PDF in this laboratory.
2. **Open** the `.pdf` file to view the detailed experiment documentation.
3. **Review** the implementation steps, screenshots, and observations.

### Video Demonstrations
Several experiments include video demonstrations showcasing the practical implementation:
- **Experiment 3-7**: Include YouTube video links for step-by-step AWS service demonstrations.
- Access these through the main README or directly from the experiment links.

### Laboratory Reports
Each experiment includes a comprehensive PDF report covering:
- **Aim**: The objective of the cloud computing experiment.
- **Theory**: Background concepts and service descriptions.
- **Implementation**: Step-by-step AWS console operations with screenshots.
- **Observations**: Results and learnings from the experiment.

## Learning Path

### Phase 1: Cloud Fundamentals
Building the foundation for cloud computing concepts and virtualization.
- **Experiment-1**: Study the NIST model and understand cloud computing characteristics.
- **Experiment-2A/2B**: Implement virtualization using VMware and VirtualBox.

### Phase 2: AWS Core Services
Hands-on implementation of fundamental AWS services.
- **Experiment-3**: Deploy and manage EC2 instances (IaaS).
- **Experiment-4**: Work with S3 storage service for object storage.
- **Experiment-5**: Host a static website using S3 (PaaS).

### Phase 3: Advanced Cloud Services
Exploring managed services, security, and edge computing.
- **Experiment-6**: Deploy WordPress using Lightsail (SaaS).
- **Experiment-7**: Configure IAM users, groups, and security policies.
- **Experiment-8**: Study fog computing concepts and applications.
